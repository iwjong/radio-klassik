# radio-klassik

A radio.garden-style web service that plays **only classical music
stations** from all over the world.

The application lives in [`radio-klassik/`](./radio-klassik).

```bash
cd radio-klassik
npm install
npm run dev
```

See [`radio-klassik/README.md`](./radio-klassik/README.md) for the full
write-up, feature list, and architecture.
