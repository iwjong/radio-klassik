# radio-klassik

A radio.garden-style web service that plays **only classical music
stations** from all over the world.

The application lives in [`klassik-globe/`](./klassik-globe).

```bash
cd klassik-globe
npm install
npm run dev
```

See [`klassik-globe/README.md`](./klassik-globe/README.md) for the full
write-up, feature list, and architecture.
